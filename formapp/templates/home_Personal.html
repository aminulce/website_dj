Finite element analysis

Finite element is implemented using javascript. Mathematical calculations are done using Math.js a javascript based open source matrix library, and the figures are drawn using plotly another open source js library. A documentation on the input file is provided in my github repository.

A sample input is provided in the following textarea. The inputs are collected using JavaScript Object Notation (JSON). For details on JSON please visit Wikipedia.

In general, the variable is declared on the left side of colon (:) and the right side of the colon contains the value that is assigned to the variable. For example,

The first line after the initial curly brace reads as "E" : 2e6. This means the value of the variable "E" is 2x10^6.

From top to bottom the variables are:

  • E - Elastic modulus.
  • nu - Poisson's ratio.
  • Nodes - The coordinates of the nodes, [During calculation, the first node is assigned a value 1, second node is assigned a value of 2 and so on.]
  • elems - The sequence of the nodes that form an element. Each row contains one element.
  • loads - Node number, direction of load application and load value is provided in sequence. Each row contains one load.
  • BC - Boundary Conditions. Node number where the BC is applied and the direction in which the movement is restricted is provided in sequence.

For details please visit github repository. Feel free to play with the inputs. For any bug report please contact @ aminul(dot)ce(at)outlook(dot)com
